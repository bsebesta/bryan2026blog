{{- /* Facet label for an index row. Takes a Page. The pipeline keeps the vault's own vocabulary, where books and films are both `type: log` (config.yaml type_by_folder) — a log is the activity, the medium is a tag. That's right for the portable layer, but "log" is opaque to a reader, so the SITE relabels it by medium: a log tagged `film` reads "movie", one tagged `book` reads "book". This lives here, not in the pipeline, so the vault vocabulary stays intact and only the presentation changes. */ -}} {{- $label := .Params.type -}} {{- if eq $label "log" -}} {{- if in .Params.tags "film" -}}{{- $label = "movie" -}} {{- else if in .Params.tags "book" -}}{{- $label = "book" -}} {{- end -}} {{- end -}} {{- with $label }}{{ . }}{{ end -}}
